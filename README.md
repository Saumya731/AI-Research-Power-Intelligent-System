# 🤖 AI Research Power Intelligent System

An AI-powered research paper search and recommendation system built using Python, Sentence Transformers, FAISS, and Hugging Face Transformers.

## 🚀 Features

- Semantic research paper search
- AI-powered paper recommendation
- Automatic paper summarization
- Fast similarity search with FAISS
- Google Colab compatible

## 🛠️ Tech Stack

- Python
- Sentence Transformers
- FAISS
- Hugging Face Transformers
- Pandas
- NumPy
- PyTorch

## 📂 Dataset

**ML-ArXiv Papers Dataset** from Hugging Face.

## ▶️ How to Run

1. Install the required libraries.
2. Run the notebook in Google Colab.
3. Load the dataset.
4. Generate embeddings.
5. Perform semantic search and summarization.
